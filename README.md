# 黎族记忆：船型屋与织绣文化展 (Li Minority Culture Exhibition)

一个展现海南黎族传统文化的沉浸式 3D 交互网站。本项目利用倾斜摄影技术复原了白查村古村落场景，并提供了高精度的文物3D模型展示与互动体验。

访问网址：https://n0mad49.github.io/li-memory/
注：加载时间可能较慢

## 🌟 项目特色

*   **沉浸式 3D 场景**：
    *   **全景漫游**：基于 3D Tiles 技术的白查村古村落点云/摄影测量模型展示。
    *   **双重视角**：支持“环绕模式”（上帝视角）和“无人机模式”（第一人称自由飞行探索）。
    *   **空间标注**：在村落中嵌入了历史文化信息点，点击即可查看图文详情。
*   **虚拟文物展馆**：
    *   高清展示黎族织锦、独木器具、藤竹编技艺、文昌灰塑等非遗文物。
    *   支持 360 度旋转观察模型细节。
*   **🎨 黎锦图案生成器**：
    *   内置的交互式创作工具，允许用户自定义生成黎族传统纹样（人形纹、蛙纹、植物纹、几何纹）。
    *   支持调整颜色、纹样类型、排列组合，实时渲染出独一无二的黎锦图案。
*   **美学设计**：
    *   采用黎族经典的红、黄、靛蓝配色体系。
    *   融合磨砂玻璃与传统纹样的 UI 设计风格。

## 🛠️ 技术栈

*   **核心引擎**: [Three.js](https://threejs.org/) (r160+)
*   **模型加载**: GLTFLoader, DRACOLoader
*   **大场景渲染**: [3d-tiles-renderer](https://github.com/NASA-AMMOS/3DTilesRendererJS) (用于加载倾斜摄影/点云数据)
*   **后期处理**: EffectComposer (UnrealBloomPass 辉光效果等)
*   **前端基础**: HTML5, CSS3 (Grid/Flex, Animations), Vanilla JavaScript (ES Modules)

## 🚀 快速开始

### 1. 运行环境
由于项目使用了 ES Modules 导入和外部 3D 模型资源，受浏览器的跨域策略 (CORS) 限制，**不能直接双击 `index.html` 打开**。

你需要一个本地 Web 服务器。

### 2. 启动方式 (推荐)

**方法 A: 使用 VS Code Live Server 插件**
1. 在 VS Code 中打开本项目文件夹。
2. 安装 "Live Server" 扩展。
3. 右键点击 `index.html`，选择 "Open with Live Server"。

**方法 B: 访问以下网址**
https://n0mad49.github.io/li-memory/
注：加载时间可能较慢

## 🎮 操作说明

### 全景场景 (白查村)
*   **环绕模式 (Orbit)**:
    *   `左键拖拽`: 旋转视角
    *   `右键平移`: 移动地图
    *   `滚轮`: 缩放
*   **无人机模式 (Drone)**:
    *   `W/A/S/D`: 前后左右飞行
    *   `Q/E`: 垂直升降
    *   `鼠标`: 调整朝向
    *   `ESC`: 退出控制

### 文物展示
*   点击右侧菜单切换不同文物。
*   选择 **“🎨 黎锦图案生成器”** 进入创作模式，调整参数后点击“生成”即可查看效果。

## ⚠️ 注意事项
*   本项目依赖网络加载 CDN 上的 Three.js 库，请确保网络连接正常。
*   部分高精度模型或点云场景可能需要较好的显卡性能支持。
*   如果遇到模型无法加载，请检查浏览器控制台 (F12) 是否有 404 错误，确认 `assets/` 路径下文件是否存在。
